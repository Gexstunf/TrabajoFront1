# Recetas App

## Descripción

Este proyecto es una aplicación web desarrollada con **Astro** que permite explorar distintas recetas utilizando una API externa.

La aplicación muestra un catálogo de comidas, permite buscar recetas y ver el detalle completo de cada una, incluyendo ingredientes, instrucciones y video.

---

## 🚀 Funcionalidades

* Buscador de recetas
* Página de detalle de cada receta
* Modo oscuro (dark mode)
* Diseño responsive (adaptado a celular y PC)
* Visualización de video de preparación
* Lista de ingredientes
* Consumo de API externa (TheMealDB)

---

## Tecnologías utilizadas

* Astro
* HTML
* CSS
* JavaScript
* API: TheMealDB

---

## Instalación y ejecución

Entrar en la carpeta del proyecto:

```
cd recetas-app
```

Instalar dependencias:

```
npm install
```

Ejecutar el proyecto:

```
npm run dev
```

Abrir en el navegador:

```
http://localhost:4321
```
